# T-006 — Cron Watchdog And Stuck-Run Control Loop

## Goal

Add a cron-safe control loop that checks active runs, detects stuck states, and starts the next run only when safe.

## Acceptance Criteria

- watchdog can inspect current run state
- watchdog can detect stale/no-output conditions
- watchdog writes back a useful note or blocker state
- watchdog can launch the next ready run when no active run exists
