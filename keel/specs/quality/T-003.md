# T-003 — Acceptance Reconciliation Before Close-Out

## Goal

Detect when task state, QA docs, and other acceptance sources disagree before marking work complete.

## Scope

- support repo-configured reconciliation checks
- compare task completion claims with known docs or acceptance files
- block final close-out when contradictions are found

## Acceptance Criteria

- a run can fail final acceptance when task state and docs disagree
- reconciliation output clearly identifies the conflicting files/claims
- successful runs without contradictions remain green
